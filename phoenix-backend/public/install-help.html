راهنمای نصب مستقیم ققنوس

اگر گوشی اجازه نصب نداد، اینجا قدم‌به‌قدم درستش کن.

بعضی گوشی‌ها موقع نصب مستقیم، پیام «نصب از منبع ناشناس» نشان می‌دهند. این یعنی باید به همان برنامه‌ای که فایل را با آن دانلود یا باز کرده‌ای، مثل Chrome یا Files یا Telegram، اجازه نصب بدهی.

نکته مهم: اگر فایل را مثلاً با Chrome دانلود کرده‌ای، باید مجوز را برای خود Chrome روشن کنی؛ اگر با Files یا Telegram یا Bazaar باز کرده‌ای، باید همان برنامه را فعال کنی، نه یک برنامه دیگر.
اگر نصب کامل شد، اپ ققنوس را باز کن و با شماره موبایل خودت وارد شو.
